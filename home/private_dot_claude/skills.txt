# Claude Code skills manifest
#
# The skill *folders* under ~/.claude/skills/ are intentionally NOT tracked by
# chezmoi (~130 MB, fetched from upstream). This file is the reproducible list:
# the source repos every installed skill came from. On a new machine run
#
#     install-claude-skills
#
# (deployed to ~/.local/bin) to clone each repo and populate ~/.claude/skills/.
#
# Format, one repo per line:
#     <github-owner/repo> [root-skill-name]
#   - The repo is shallow-cloned and every SKILL.md directory inside it is
#     copied into ~/.claude/skills/, keeping its folder name.
#   - If a repo's SKILL.md sits at the repo root, the whole repo is copied and
#     named <owner>_<repo>, unless an explicit [root-skill-name] is given.
#   - Lines starting with # and blank lines are ignored.
#
# Archived skills: folder names listed in ARCHIVED.md (deployed to
# ~/.claude/ARCHIVED.md) are skipped by install-claude-skills, so the mixed
# collections below only repopulate the skills actually in use.
